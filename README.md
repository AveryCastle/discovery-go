## Local setting
To set GOPATH enviroment variable to point to the location,

  `$ export GOPATH=$HOME/workspace/discovery-go`
  
Next, to compile it,

  `$ go install github.com/JuHyun/discovery-go/exercise1.6.3`

Next, to execute it,  
  
  `$ GOPATH/bin/exercise1.6.3`
